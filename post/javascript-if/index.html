자바스크립트의 if 사용법

기본 사용법

if만 사용하는 방법

기본적인 구문은 아래와 같다.

if (condition) {
   statement
}

if 구문의 condition이 참일 경우 statement문을 실행한다.

아래의 예시를 통해 확인해보자.

입력 받은 값이 숫자인지 판별하는 로직을 if만으로 작성한 예시

const inputNumber = Number(prompt("숫자를 입력하세요."));

// isNaN는 입력 받은 값이 NaN인지 검사하는 함수
if (isNaN(inputNumber)) {
  console.log("숫자를 입력하세요.");
}

// isNaN는 입력 받은 값이 NaN인지 검사하는 함수
if (!isNaN(inputNumber)) {
  console.log("숫자를 입력하셨습니다.");
}

하지만 굳이 isNaN과 !isNaN을 따로 검사하기 보다는

if와 else와 같이 사용하는 방법

기본적인 구문은 아래와 같다.

if (condition) {
   statement1
} [else {
   statement2
}]

if 구문의 condition이 참일 경우 statement1을 실행한다. 만약 else문이 존재하고 if 구문의 condition이 거짓일 경우 statement2를 실행한다.

아래의 예시를 통해 확인해보자.

입력 받은 값이 숫자인지 판별하는 로직을 if와 else를 사용해서 작성한 예시

const inputNumber = Number(prompt("숫자를 입력하세요."));

// isNaN는 입력 받은 값이 NaN인지 검사하는 함수
if (isNaN(inputNumber)) {
  console.log("숫자를 입력하세요.");
} else {
  console.log("숫자를 입력하셨습니다.");
}

if와 else if와 else와 같이 사용하는 방법

기본적인 구문은 아래와 같다.

if (condition1)
   statement1
else if (condition2)
   statement2
else if (condition3)
   statement3
...
else
   statementN

if 구문의 condition1이 참일 경우 statement1을 실행한다. condition1가 참이 아닌 경우 condition2를 확인, condition2가 참인 경우 statement2를 실행, condition2가 참이 아닌 경우 condition3를 확인, condition3가 참인 경우 statement3를 실행, …, 마지막 else if문 까지 확인을 한 뒤 모두 참이 아닌 경우는 else 문의 statementN을 실행한다.

아래의 예시를 통해 확인해보자.

생성한 난수의 속한 범위를 출력하는 로직을 if, else if와 else를 사용해서 작성한 예시

// Math.random은 난수를 생성하는 함수
const randomNumber = Math.floor(Math.random() * 100);

if (randomNumber < 5) {
  console.log("랜덤으로 생성된 숫자는 5 미만입니다.");
} else if (randomNumber < 10) {
  console.log("랜덤으로 생성된 숫자는 5 이상 이거나 10 미만입니다.");
} else if (randomNumber < 15) {
  console.log("랜덤으로 생성된 숫자는 10 이상 이거나 15 미만입니다.");
} else if (randomNumber < 20) {
  console.log("랜덤으로 생성된 숫자는 15 이상 이거나 20 미만입니다.");
} 
    ... 
} else {
  console.log("랜덤으로 생성된 숫자는 95 이상 이거나 100 미만입니다.");
}
