javascript - if

  2022-04-05       javascript 

개발자의 if문은 격투가의 정권 찌르기와 같다. 가장 기초적인 움직임으로써 초보 개발자와 고수 개발자의 차이는 if문을 얼마나 잘 사용하는가에 따라 갈린다고 볼 수 있다.

이 문서에서는 if문의 기본 사용법부터 시작해서 응용 사용법까지 정리해두었다. 세상의 모든 사장님들에게 정권 찌르기를 멋지게 찌를 수 있는 그 날 까지 if문을 갈고 닦아 보도록하자.

기본 사용법

대검의 고수는 발도와 납도보면 알 수 있다.

– 몬스터 헌터 중독자 코더 신씨

if를 사용하는 방법

기본적인 구문은 아래와 같다.

if (condition) {
   statement
}

if 구문의 condition이 참일 경우 statement문을 실행한다.

아래의 예시를 통해 확인해보자.

예시) 입력 받은 값이 숫자인지 판별하는 로직

const inputNumber = Number(prompt("숫자를 입력하세요."));

// isNaN는 입력 받은 값이 NaN인지 검사하는 함수
if (isNaN(inputNumber)) {
  console.log("숫자를 입력하세요.");
}

// isNaN는 입력 받은 값이 NaN인지 검사하는 함수
if (!isNaN(inputNumber)) {
  console.log("숫자를 입력하셨습니다.");
}

하지만 위의 로직은 isNaN과 !isNaN을 따로 두 번 검사하기 때문에 비효율적이다.

보다는 isNaN을 한번만 검사 후 isNaN이 참인 경우 statement을 그렇지 않은 경우 다른 statement를 실행하는 것이 효율적이다. 아래의 예시를 확인해보자.

if와 else와 같이 사용하는 방법

검과 총, 어느 것을 선택할지 망설이기 보다는 건랜스를 들어라.

– 몬스터 헌터 중독자 코더 신씨

기본적인 구문은 아래와 같다.

if (condition) {
   statement1
} else {
   statement2
}

if 구문의 condition이 참일 경우 statement1을 실행한다. 그렇지 않은 경우 statement2를 실행한다.

아래의 예시를 통해 확인해보자.

예시) 입력 받은 값이 숫자인지 판별하는 로직

const inputNumber = Number(prompt("숫자를 입력하세요."));

// isNaN는 입력 받은 값이 NaN인지 검사하는 함수
if (isNaN(inputNumber)) {
  console.log("숫자를 입력하세요.");
} else {
  console.log("숫자를 입력하셨습니다.");
}

만약 입력 받은 숫자의 범위를 판별하고 싶은 경우는 위의 방법만으로는 충분하지 않다. 이 경우는 아래의 예시를 확인해보자.

if와 else if와 else와 같이 사용하는 방법

이거 누가 else if를 이렇게나 많이 만들었어? 어라 왜 git log에 내 이름이?

– 현실을 부정하고 싶은 코더 신씨

기본적인 구문은 아래와 같다.

if (condition1)
   statement1
else if (condition2)
   statement2
else if (condition3)
   statement3
...
else
   statementN

if 구문의 condition1이 참일 경우 statement1을 실행한다. condition1가 참이 아닌 경우 condition2를 확인, condition2가 참인 경우 statement2를 실행, condition2가 참이 아닌 경우 condition3를 확인, condition3가 참인 경우 statement3를 실행, …, 마지막 else if문 까지 확인을 한 뒤 모두 참이 아닌 경우는 else 문의 statementN을 실행한다.

아래의 예시를 통해 확인해보자.

생성한 난수의 속한 범위를 출력하는 로직을 if, else if와 else를 사용해서 작성한 예시

// Math.random은 난수를 생성하는 함수
const randomNumber = Math.floor(Math.random() * 100);

if (randomNumber < 5) {
  console.log("랜덤으로 생성된 숫자는 5 미만입니다.");
} else if (randomNumber < 10) {
  console.log("랜덤으로 생성된 숫자는 5 이상 이거나 10 미만입니다.");
} else if (randomNumber < 15) {
  console.log("랜덤으로 생성된 숫자는 10 이상 이거나 15 미만입니다.");
} else if (randomNumber < 20) {
  console.log("랜덤으로 생성된 숫자는 15 이상 이거나 20 미만입니다.");
} 
    ... 
} else {
  console.log("랜덤으로 생성된 숫자는 95 이상 이거나 100 미만입니다.");
}
