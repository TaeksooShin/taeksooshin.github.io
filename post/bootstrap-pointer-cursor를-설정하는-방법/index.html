bootstrap - pointer cursor를 설정하는 방법

  2022-04-07       bootstrap 

bootstrap을 사용하고 있음에도 불구하고 잔뜩 CSS를 직접 수정하는 무지성 코딩을 하던 코더 신씨, 그의 무서운 선배는 오늘도 코더 신씨의 코드를 보며 한숨을 내쉬었다.

“CSS로 직접 cursor: pointer를 지정하는 짓 좀 그만해라. bootstrapper는 bootstrapper 답게 bootstrap 안에서 해결하도록 해!”

난 분명 프론트로 취직했는데 bootstrapper 취급받는 차디찬 현실에 눈물을 흘리며, CSS를 만지지 않고 cursor를 pointer로 변경하는 방법을 찾고 있는 코더 신씨, 잘 찾아왔다. 그런 당신을 위해 준비한 문서다.

role=“button"을 지정해서 hover시 cursor의 모양을 pointer로 변경하기

bootstrap 버전 4에서 추가되었다고 하는데 (확인 필요) role=“button” 속성을 지정하면 hover시에 cursor 모양이 pointer로 변경된다고 한다. 예시를 확인해보자.

예시) role=“button"을 지정해서 hover시 cursor의 모양을 pointer로 변경

<span role="button">
  hover시에 cursor가 pointer가 되는 span입니다.
</span>

CSS 상에서 cursor: pointer를 직접 지정해서 hover시 cursor의 모양을 pointer로 변경하기

혹시 bootstrap 버전이 낮아서 위의 대응이 되지 않는 사람들을 위해 준비한 코너. 바로 예시를 확인해보자.

예시) CSS 상에서 cursor: pointer를 직접 지정해서 hover시 cursor의 모양을 pointer로 변경

<span class="cursor-pointer-on-hover">
  hover시에 cursor가 pointer가 되는 span입니다.
</span>

.cursor-pointer-on-hover:hover {
  cursor: pointer;
}

결론

만약 bootstrap 버전이 2일 경우는 눈물을 흘리며 cursor: pointer를 직접 CSS로 지정하도록 하자. 아니면 버전을 올리도록 하자. 만약 버전을 올리는 것을 허락하지 않는다면 회사에 사표를 내고 bootstrapper에서 탈출할 계획을 세우도록 하자.
