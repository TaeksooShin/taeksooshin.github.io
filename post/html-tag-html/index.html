html - html 태그

  2022-04-06       html 

html 태그는 문서의 루트(최상단 요소)를 나타낸다.

사용할 수 있는 속성

속성 속성 값 예시 설명
lang ko, ja, en 현재 문서가 무슨 언어로 구성되어 있는지 설명
xmlns http://www.w3.org/1999/xhtml xml 구문 분석기를 이용할 경우는 필수

내부에 넣을 수 있는 태그

태그 간단 설명
[[html tag-head]]{head} 문서 정보(메타데이터)를 담는다
[[html tag-body]]{body} 문서의 내용을 나타냄

주의사항

  • 모든 다른 요소는 의 요소의 후손이어야 한다.
  • lang은 스크린 리더가 음성 표현에 사용할 언어를 선택할 때 도움이 되기 때문에 반드시 사용하도록 하자
