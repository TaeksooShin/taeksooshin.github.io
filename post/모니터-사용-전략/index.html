모니터 사용 전략

개요

많은 모니터를 쓰더라도 창이동을 피할 수 없는데, 최대한 창이동을 하지 않고 사용하기 위한 나의 노력을 기록해 봤다.

집에서 작업을 할 때에는 트리플 모니터(27인치 한대, 23인치 두대 + 별도의 노트북), 회사에서는 듀얼 모니터(맥북 16인치, 23인치 한대 + 별도의 노트북)를 사용하고 있다.

트리플 모니터

27인치 모니터

  • 주로 사용하는 모니터.
  • 모든 작업이 여기서 이루어진다.
  • 검색을 할 때는 웹브라우저를 여기로 가지고 와서 검색을 한 후에 원래의 장소로 되돌린다.
  • 일정표, TODO 리스트도 작성할 필요가 있는 것은 여기로 가져와서 작업하고 원래의 장소로 되돌린다.
  • 커서가 거의 여기에 있음으로 가끔씩 클릭이 필요한 스포티파이도 여기에 둔다.

23인치 모니터

  • 주로 결과물 확인 하는 모니터.
  • 취미로 코딩을 할 때에는 유튜브, 넷플릭스를 시청하는 용도로 사용 하기도 함

틸트 23인치 모니터

  • 틸트를 해서 세로를 길게 사용하는 모니터.
  • 주로 일정을 관리하기 + 레퍼런스를 참고 하기 위해서 사용하는 모니터
  • 모니터를 위 아래 반으로 나눈다고 생각하고 위에는 일정표(icalendar), 아래에는 todo용 org 모드의 emacs를 실행 시켜둔다.
  • 단 일정표의 경우는 회사 사람들과 약속을 동기화 해서 사용하는 용도만 icalendar를 사용하고 개인적인 일정은 별도의 노트북에 기록함(클라우드를 별로 신용하지 않음)
  • 레퍼런스를 참고 해서 코드를 작성할 때에 여기에 두면 세로로 길어서 많은 내용을 참고 할 수 있어서 편리하게 사용 가능해 주로 여기에 두는 편.

듀얼 모니터

23인치 모니터(4k)

  • 트리플 모니터의 27 모니터와 동일한 포지션.
  • 개인적인 의견으로는 4k는 코드를 작성할 때 별로 필요가 없는 것 같다.

16인치 모니터

  • 23인치 모니터들 두개가 합쳐진 포지션.

결론

창이동하는 시간이 아까워서 모니터 세대 샀다가, 각각의 모니터의 사용법이 중구난방으로 사용하다가 혼란이 와서 정리겸 작성한 글
