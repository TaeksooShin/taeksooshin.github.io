사용처

배열 합치기

전개 구문(spread operator)를 이용하면 배열을 합치는 Array.prototype.concat()를 대신해서 배열을 간단하게 합칠 수 있다.

const array1 = [1, 2, 3];
const array2 = [4, 5, 6];

const combinedArrayWithConcat = array1.concat(array2);
console.log(combinedArrayWithConcat); // [1, 2, 3, 4, 5, 6]

const combinedArrayWithSpread = [...array1, ...array2];
console.log(combinedArrayWithSpread); // [1, 2, 3, 4, 5, 6]

단 개인적인 의견으로는 배열을 합칠 때에는 spread operator를 사용하기 보다는 concat을 사용하는 것이 좋다고 생각한다. 왜냐하면 spread operator 자체는 배열을 합친다는 의미가 없기 때문이다. 즉 명시적으로 배열을 합친다는 의미를 가진 concat과 다르게 spread operator는 한번 더 확인 할 필요가 있기 때문에 가독성이 조금이라도 더 오를 수 있는 concat을 사용하는게 더욱 좋다고 생각된다.
