DDL

  2023-02-17       데이터베이스(database) 

용어 설명

나의 해석

DDL(Data Defintion Language)은 데이터베이스 및 테이블을 생성, 삭제 하기 위해 사용하는 SQL문이다.

특징으로는 WHERE문을 사용 할 수 없다.

각종 설명

DDL(데이터 정의 언어)은 데이터를 저장하는 데이터베이스 및 테이블을 생성 삭제하기 위한 것이다.

(SQL 더 쉽게 더 깊게 p.15)

간략 설명

CREATE : 생성 DROP : 삭제 ALTER : 변경

상세 설명

CREATE

CREATE문은 데이터베이스, 테이블을 작성할 때 사용하는 [[SQL]]문이다. 기본 구문은 아래와 같다.

데이터베이스를 생성하는 CREATE DATABASE문

CREATE DATABASE <데이터베이스명>

테이블을 생성하는 CREATE TABLE문

CREATE TABLE <테이블>
(<열명1> <데이터형> <열의 제약>,
 <열명2> <데이터형> <열의 제약>,
 <열명3> <데이터형> <열의 제약>,
 ...
 <테이블의 제약1>,
 <테이블의 제약2>,
 <테이블의 제약3>,
 ...);
