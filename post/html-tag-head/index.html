html - head 태그

  2022-04-06       html 

head 태그는 문서 정보(메타데이터)를 담는 태그로 제목, 스크립트, 스타일 시트 등을 등을 사용 가능

사용할 수 있는 속성

전역 특성만 사용 가능

내부에 넣을 수 있는 태그

태그 간단 설명
[[html tag-meta]]{meta} 웹 페이지에 추가 정보를 전달
[[html tag-title]]{title} 웹 페이지의 제목
[[html tag-script]]{script} 웹 페이지에 스크립트(js)를 추가
[[html tag-link]]{link} 웹 페이지에 다른 파일(css, favicon..등)을 추가
[[html tag-style]]{style} 웹 페이지에 스타일 정보를 포함
[[html tag-base]]{base} 웹 페이지에 기본 경로를 지정
[[html tag-noscript]]{noscript} 스크립트 유형을 지원하지 않는 경우 사용할 HTML 구획 정의

주의 사항

  • html 문서보다 먼저 호출할 필요가 없는 script의 경우는 head에 위치시키지 말고 body의 끝에 두자
