DML

  2023-02-21       데이터베이스(database) 

용어 설명

나의 해석

DML(Data Manipulation Language)은 테이블의 행을 조작하는 SQL문이다.

특징으로는 WHERE문을 사용 할 수 있다.

각종 구문

DML(데이터 조작 언어)은 테이블의 행을 검색하거나 변경하기 위한 것이다.

(SQL 더 쉽게 더 깊게 p.15)

종류

INSERT : 등록 SELECT : 조회 UPDATE : 갱신 DELETE : 삭제

상세 설명

SELECT

SELECT 문은 테이블에서 데이터를 꺼낼 때 사용하는 [[SQL]]문이다. 기본 구문은 아래와 같다.

SELECT <열명>, ...
  FROM <테이블명>;

SELECT문에 대한 각종 구문

SELECT 문은 많은 SQL 문 중에서도 가장 자주 사용되며 가장 기본이 되는 구문이다. SELECT 문을 마스터하는 것은 곧 SQL 문 전체를 마스터하는 것이라 해도 과언이 아니다. (SQL 더 쉽게 더 깊게 p.37)
