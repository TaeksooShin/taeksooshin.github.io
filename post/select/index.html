기본 사용법

SELECT 문은 테이블에서 데이터를 꺼낼 때 사용하는 [[SQL]]문이다. 기본 구문은 아래와 같다.

SELECT <열명>, ...
  FROM <테이블명>;

모든 열을 출력하고 싶은 경우

모든 열을 출력하고 싶으면 별표(*)를 사용하면 된다.

SELECT *
  FROM <테이블명>;

단 굉장히 많은 데이터가 담겨져 있는 테이블에서 위의 구문을 실행하면 리소스를 굉장히 사용하기 때문에 위의 구문을 사용하기 보다는 필요한 열만을 지정해서 출력하도록 하자. 혹은 limit를 지정해서 일정량의 데이터만 가져오도록 하자.

중복을 제외하고 출력하기

DISTINCT 키워드를 이용하면 중복을 제외한 결과를 출력한다.

SELECT DISTINCT <열 명>
  FROM <테이블명>;

복수의 열을 조합한 경우, 조합한 결과가 동일한 행이 하나의 행으로 모아져 출력된다.

DISTINCT를 사용할 때 NULL도 하나의 데이터로 인식된다.

WHERE

WEHRE문을 이용하면 조건부에 부합하는 행만을 출력할 수 있다. 구문은 아래와 같다.

SELECT <열명>, ...
  FROM <테이블명>
 WHERE <조건식>;

WHERE 구는 FROM 구 바로 뒤에 둔다.

각종 인용

SELECT 문은 많은 SQL 문 중에서도 가장 자주 사용되며 가장 기본이 되는 구문이다. SELECT 문을 마스터하는 것은 곧 SQL 문 전체를 마스터하는 것이라 해도 과언이 아니다. (SQL 더 쉽게 더 깊게 p.37)
