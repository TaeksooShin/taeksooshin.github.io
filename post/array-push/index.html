배열의 마지막에 요소 추가하기(array push)

  2023-03-15       프로그래밍 언어(programming language) 

[[javascript]]{자바스크립트(JavaScript)}

자바스크립트에서 배열의 마지막에 요소를 추가하기 위해서 사용하는 방법은 Array.prototype.push() 메서드를 사용하는 방법과 [[spread operator]]{전개 구문(spread operator)}를 사용하는 방법 크게 두 가지가 있다.

Array.prototype.push() 메서드를 사용하는 방법은 아래와 같다.

배열의 마지막에 요소 추가하기 - Array.prototype.push()

const array = [1, 2, 3];
array.push(4);

console.log(array); // [1, 2, 3, 4]

[[spread operator]]{전개 구문(spread operator)}를 사용하는 방법은 아래와 같다.

배열의 마지막에 요소 추가하기 - [[spread operator]]{전개 구문(spread operator)}

const array = [1, 2, 3];
const pushedArray = [...array, 4];

console.log(pushedArray); // [1, 2, 3, 4]
console.log(array);       // [1, 2, 3]
