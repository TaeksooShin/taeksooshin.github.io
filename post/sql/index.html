SQL(Structured Query Language)란?

나의 해석

데이터베이스를 조작하기 위한 특정 도메인 언어.

각종 설명

SQL(Structured Query Language)은 관계형 데이터베이스(이하, 데이터베이스)를 제어하기 위한 언어다.

(SQL 더 쉽게 더 깊게 p.14)

SQL문의 종류

SQL문의 종류는 세 가지로 분류한다.

  • [[DDL]]{DDL(Data Defintion Language)}
  • [[DML]]{DML(Data Manipulation Language)}
  • [[DCL]]{DCL(Data Control Language)}

각각의 내용은 각가의 링크를 참조할 것.
