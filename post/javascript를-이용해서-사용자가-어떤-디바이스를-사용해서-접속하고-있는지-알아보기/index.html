javascript를 이용해서 사용자가 어떤 디바이스를 사용해서 접속하고 있는지 알아보기

  2023-02-17       자바스크립트(javascript) 

navigator.userAgent를 이용하는 방법

navigator.userAgent는 user agent를 string으로 보관하고 있는 읽기 전용 property이다. 이를 이용해서 사용자가 pc에서 페이지를 열람하고 있는가, 스마트폰에서 페이지를 열람하고 있는가를 확인할 수 있다.

const isSmartPhone = (): boolean => {
    return /Android|iPhone/i.test(navigator.userAgent);
};

간편하게 이용할 수 있는 반면 아래와 같은 단점이 있다.

  • 사용자가 navigator.userAgent를 위조가 가능하다.
  • 호환성이 좋지 않다. (참고 - https://caniuse.com/mdn-api_navigator_useragentdata)

특히 호환성이 굉장히 좋지 않기 때문에 사용하기 전에 검토를 해볼 필요가 있다.

detectmobilebrowsers에서 제공하는 스크립트를 이용하는 방법.

http://detectmobilebrowsers.com/ 는 페이지를 어떠한 환경에서 작동하고 있는지 감지 할 수 있는 스크립트를 제공하는 사이트이다.

편하게 스크립트를 받아서 자신의 서비스에 넣어주면 편리하게 사용가능하다.

단 몇몇 기기에서 작동이 되지 않는다는 제보도 있기 때문에 100% 신용은 금물이다.

팁
