header

  Jun 8, 2021       html 

개요

header는 소개 및 네비게이션에 대한 정보를 담는 태그

예제

  1. 단순한 header 사용 방법
<header>
 <p>Welcome to...</p>
 <h1>Voidwars!</h1>
</header>

출력결과

Welcome to...

Voidwars!

  1. hgroup을 이용한 heading 집약
<header>
 <hgroup>
  <h1>Fullscreen API</h1>
  <h2>Living Standard — Last Updated 19 October 2015</h2>
 </hgroup>
 <dl>
  <dt>Participate:</dt>
  <dd><a href="https://github.com/whatwg/fullscreen">GitHub whatwg/fullscreen</a></dd>
  <dt>Commits:</dt>
  <dd><a href="https://github.com/whatwg/fullscreen/commits">GitHub whatwg/fullscreen/commits</a></dd>
 </dl>
</header>

출력결과

Fullscreen API

Living Standard — Last Updated 19 October 2015

Participate:
GitHub whatwg/fullscreen
Commits:
GitHub whatwg/fullscreen/commits

설명

  • heading(h1 - h6)를 반드시 포함할 필요는 없다.
  • nav 태그를 포함해도 된다. (디자인적 계획에 따라서 포함하지 않아도 괜찮다.)
  • footer, address, 다른 header는 포함할 수 없다.
