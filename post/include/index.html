include

  2023-03-07       프로그래밍 언어(programming language) 

[[array]]{배열}에 특정 값이 존재하는지 확인하기

[[javascript]]{JavaScript}

includes 함수를 사용하면 된다.

includes 함수의 사용법은 아래와 같다.

대상_배열.includes(특정_값);

구체적인 사용법

const animals = ['cat', 'dog']

console.log(animals.includes('dog')); // true
console.log(animals.includes('lion')); // false

팁

[[javascript]]{JavaScript} - 이전까지 사용한 indexOf를 사용해도 괜찮지 않을까?

결론부터 이야기 하자면, includes 함수를 사용하는 것이 훨씬 좋다. 왜냐하면 가독성적인면에서 훨씬 좋기 때문이다. includes로 작성된 코드를 보면 배열에 특정 값을 포함하고 있는가를 확인하고 있는 코드라는 것을 바로 알 수 있다. 하지만 indexOf를 이용하는 경우는? 무슨 의도로 이러한 코드를 작성했는지 면밀히 봐야지 코드 작성자의 의도를 확인 할 수 있다. 그렇기 때문에 includes 함수를 이용하는 것이 훨씬 좋다.
